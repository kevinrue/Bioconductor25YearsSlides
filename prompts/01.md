## Original Goal

This repository should:

- include a folder where contributors add photos through pull requests
- include a GitHub Action that deploys a rotating photo carousel to GitHub Pages
- include a banner at the top of the page telling viewers where to contribute

## Requirements

- Display photos in alphabetical filename order
- Be robust to varying photo dimensions
- If possible, check image size/resolution in pull requests
- Prefer R or Python (no hard language requirement)
- Include local preview instructions in README
- Auto-advance the carousel every 10 seconds

## Implemented Solution (Current State)

- `photos/` is the contributor upload folder
- `site/` contains the static carousel frontend
- `scripts/generate_manifest.py` generates `photos/manifest.json` from photo filenames
- `scripts/check_images.py` validates image byte-size and dimensions
- `.github/workflows/check-images.yml` runs PR checks on contributed photos
- `.github/workflows/deploy-pages.yml` builds and deploys to GitHub Pages
- The page banner links contributors to the repo pull requests page
- The carousel auto-advances every 10 seconds (`ROTATION_MS = 10000`)
